чтобы запустить СКРИПТ и чтобы он работал без ошибок для этого нужен:
1. Иметь python  3.9 чтобы скрипт стабильно запускался,
2. Установить библотеку instabot в cmd пишите pip install instabot,
3. Чуть не забыл после каждого запуска желательно удалять файл с именем config. 
Все если сделали правильно то можете запускать батник.
